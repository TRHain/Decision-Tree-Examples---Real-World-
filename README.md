# Decision-Tree-Examples---Real-World-
This is a simple example of a real world decision tree and why they are important. This is not connected to any API as that would be the next step. 

This project was created as a discussion topic for PY101 at Maestro College. The assignment required us to utilize a decision tree and explain how it is used, 
where it is effective, and where limitations or failure points may occur.

This example uses a person living in New York City who needs to make an informed transportation decision. 
The decision tree evaluates factors such as weather conditions, distance, available transportation methods, 
time constraints, and other real-world issues that may arise. The order of these checks matters because some 
factors, such as safety and accessibility, can immediately eliminate certain transportation options before smaller 
preferences are considered.
